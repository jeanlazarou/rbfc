_RBFC_ is a set of tools for the _brainfuck_ language.

### Compile a program:
```
bin/rbfc <program-file>
```

### Run a program:
```
bin/rbfr <program-file>
```

### Debug a program:
```
bin/rbfd <program-file>
```

### Transpile to Javascript:
```
bin/bftojs <program-file>
```
